> [!list]
> **Традиционный RAG, основанный только на векторах:**
> - Встраивание документов → сохранение в векторной БД
> - Встраивание запроса → поиск похожих векторов
> - Извлечение исходного текста → передача в LLM
> 
> **Гибридный RAG (рекомендуется):**
> - Индексирование и текста (BM25), и векторов (семантический поиск)
> - Поиск по запросу одновременно по ключевым словам и по семантике
> - Комбинирование оценок с помощью методов вроде Reciprocal Rank Fusion
> - Передача лучших результатов в LLM
