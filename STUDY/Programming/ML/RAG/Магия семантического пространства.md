
> [!attention] 
> `Embeddings` создают **мульти-размерное пространство**, где
> - Похожие концепты располагаются близко
> - Сохраняется контекст
> - Сохраняется взаимосвязь 

```python
from sentence_transformers import SentenceTransformer  
  
model = SentenceTransformer('all-MiniLM-L6-v2')  
  
# Create embeddings  
texts = [  
    "The dog is running in the park",  
    "A canine is jogging outdoors",   
    "The car needs engine repair",  
    "Dogs love to play outside"  
]  
  
embeddings = model.encode(texts)  
print(f"Each embedding has {embeddings.shape[1]} dimensions")  
# Output: Each embedding has 384 dimensions  
  
# Calculate similarities  
from sklearn.metrics.pairwise import cosine_similarity  
similarities = cosine_similarity(embeddings)  
print("Similarity matrix:")  
print(similarities)  
# You'll see sentences 1, 2, and 4 are more similar to each other
```
