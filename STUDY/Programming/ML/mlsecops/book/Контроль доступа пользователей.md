> [!security]
> Ограничение доступа пользователей снижает риск несанкционированного проникновения:  
> - Использовать сильные уникальные пароли для всех аккаунтов  
> - Отключить root-логин; для администрирования использовать `sudo`  
> - Регулярно проверять список пользователей и удалять/блокировать неиспользуемые
