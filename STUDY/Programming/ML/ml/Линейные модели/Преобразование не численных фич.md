
> [!problem] 
> А что делать, если некоторые фичи принимают какие-то конкретные не численные, `категориальные`, значения?

> [!solved] 
> Первым решением данной задачи может быть `one-shot encoding`, то есть дробление фичи в N фич по кол-ву категорий с бинарным признаком принадлежности каждой из них. 

> [!attention] 
> Одна из фич закономерно выпадет в силу регуляризации, чтобы избежать переобучения.
