
> [!question] 
> Как теперь научиться оптимизировать w так, чтобы модель как можно лучше предсказывала логиты? Нужно применить метод максимума правдоподобия для распределения Бернулли. Это самое простое распределение, которое возникает, к примеру, при бросках монетки, которая орлом выпадает с вероятностью p. У нас только событием будет не орёл, а то, что пользователь кликнул на объект с такой вероятностью.

> [!done] 
> Правдоподобие позволяет понять, насколько вероятно получить данные значения таргета y при данных X и весах w. Оно имеет вид:
> $$p(y\ |\ X,w)=\prod_i p(y_i\ |\ x_i,w)$$
> А для распределения Бернулли получается:
> $$p(y\ |\ X,w) = \prod_i p_i^{y_i}(1-p_i)^{1-y_i},$$
> $p_i$ - вероятность, посчитанная из ответов модели.
